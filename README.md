EV Fleet Optimizer
This project is dedicated to the optimization of electric vehicle fleet management. It focuses on analyzing energy efficiency, charging cycles, and battery performance. The work is developed within the framework of the Master's program in Electromobility at Friedrich-Alexander-University Erlangen-Nuremberg (FAU).

Project Overview
The tool provides a computational approach to fleet management by integrating electrical engineering principles with data science. It is designed to process technical parameters of EVs and provide optimization strategies for charging infrastructure.

Technical Specifications
Language: Python 3.10+

Environment: Anaconda / VS Code

Main Libraries: TensorFlow, Scikit-learn, NumPy, Pandas

Project Structure
my_web.py: Main application script.

requirements.txt: List of dependencies.

static/ and templates/: Web interface files and assets.

Installation and Setup
Clone the repository:
git clone https://github.com/93sur/ev-fleet-optimizer.git

Activate the environment:
conda activate TensorflowEnviro

Install requirements:
pip install -r requirements.txt

Academic Context
The project incorporates:

Thermal management analysis based on caloric state variables.

Electrical circuit modeling and AC analysis.

Machine learning architectures for predictive analytics.
